# Suppression de nœuds, de contenu et de valeurs d&#39;un document XML
Une fois qu'un DOM \(Document Object Model\) XML est en mémoire, vous pouvez supprimer des nœuds de l'arborescence ou supprimer du contenu et des valeurs de certains types de nœuds.  Pour plus d'informations sur la suppression d'un nœud sans descendant ou d'une sous\-arborescence entière d'un document, voir [Suppression des nœuds du DOM](../../../../docs/standard/data/xml/removing-nodes-from-the-dom.md).  Pour plus d'informations sur la suppression d'attributs d'un élément, voir [Suppression d'attributs d'un nœud d'élément dans le DOM](../../../../docs/standard/data/xml/removing-attributes-from-an-element-node-in-the-dom.md).  Pour plus d'informations sur la suppression de contenu d'un nœud sans supprimer le nœud, voir [Suppression du contenu d'un nœud dans le DOM](../../../../docs/standard/data/xml/removing-node-content-in-the-dom.md).
