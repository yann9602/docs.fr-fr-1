Celle-ci relève de la responsabilité du wrapper.</span><span class="sxs-lookup"><span data-stu-id="71e8b-145">Because the wrapper is potentially exposing callers to these resources, careful verification of the safety of the native code is necessary and is the wrapper's responsibility.</span></span>
