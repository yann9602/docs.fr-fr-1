<span data-ttu-id="4eb07-103">Un accesseur add ou remove doit avoir un corps</span><span class="sxs-lookup"><span data-stu-id="4eb07-103">An add or remove accessor must have a body</span></span>  
  
 <span data-ttu-id="4eb07-104">Un mot clé **add** ou **remove** dans une définition d’ [événement](../../csharp/language-reference/keywords/event.md) doit avoir un corps.</span><span class="sxs-lookup"><span data-stu-id="4eb07-104">An **add** or **remove** keyword in an [event](../../csharp/language-reference/keywords/event.md) definition must have a body.</span></span> <span data-ttu-id="4eb07-105">Pour plus d’informations, consultez [Événements](../../csharp/programming-guide/events/index.md).</span><span class="sxs-lookup"><span data-stu-id="4eb07-105">For more information, see [Events](../../csharp/programming-guide/events/index.md).</span></span>  
  
 <span data-ttu-id="4eb07-106">L’exemple suivant génère l’erreur CS0073 :</span><span class="sxs-lookup"><span data-stu-id="4eb07-106">The following sample generates CS0073:</span></span>  
  
```  
// CS0073.cs  
delegate void del();  
  
class Test  
{  
   public event del MyEvent  
   {  
      add;   // CS0073  
      // try the following lines instead  
      // add  
      // {  
      //    MyEvent += value;  
      // }  
      remove  
      {  
         MyEvent -= value;  
      }  
  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
