 <span data-ttu-id="3ad62-108">La séquence suivante de compilations provoque une erreur CS0012 :</span><span class="sxs-lookup"><span data-stu-id="3ad62-108">The following sequence of compilations will result in CS0012:</span></span>

```  
  
 <span data-ttu-id="3ad62-109">Puis :</span><span class="sxs-lookup"><span data-stu-id="3ad62-109">Then:</span></span>  
  
```

{  
   public static A f()  
   {  
      return new A();  
   }  
}  
```  
  
 <span data-ttu-id="3ad62-110">Puis :</span><span class="sxs-lookup"><span data-stu-id="3ad62-110">Then:</span></span>

   public static void Main()  
   {  
      object o = B.f();   // CS0012  
   }  
}  
```
