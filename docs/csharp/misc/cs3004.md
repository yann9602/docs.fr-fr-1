# Avertissement du compilateur (niveau&#160;1) CS3004
Les caractères Unicode mélangés et décomposés ne sont pas conformes CLS  
  
 Seuls les caractères UNICODE composés sont autorisés dans les identificateurs [public](../../csharp/language-reference/keywords/public.md), [protected](../../csharp/language-reference/keywords/protected.md) ou `protected` `internal` afin d’être conformes à la spécification CLS \(Common Language Specification\). Pour plus d’informations sur la conformité CLS, consultez [Écriture d’un code conforme CLS](http://msdn.microsoft.com/fr-fr/4c705105-69a2-4e5e-b24e-0633bc32c7f3) et [Indépendance du langage et composants indépendants du langage](../Topic/Language%20Independence%20and%20Language-Independent%20Components.md).
