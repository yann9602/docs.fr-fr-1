## <a name="example"></a><span data-ttu-id="ad46d-106">Exemple</span><span class="sxs-lookup"><span data-stu-id="ad46d-106">Example</span></span>  
 <span data-ttu-id="ad46d-107">L’exemple suivant génère l’avertissement CS3002 :</span><span class="sxs-lookup"><span data-stu-id="ad46d-107">The following example generates CS3002:</span></span>  
  
```  
// CS3002.cs  
  
[assembly:System.CLSCompliant(true)]  
public class a  
{  
    public ushort bad()   // CS3002, public method  
    {  
        ushort a;  
        a = ushort.MaxValue;  
        return a;  
    }  
  
    private ushort OK()   // OK, private method  
    {  
        ushort a;  
        a = ushort.MaxValue;  
        return a;  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
