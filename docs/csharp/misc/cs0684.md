# <a name="compiler-warning-level-1-cs0684"></a><span data-ttu-id="88db6-102">Avertissement du compilateur (niveau 1) CS0684</span><span class="sxs-lookup"><span data-stu-id="88db6-102">Compiler Warning (level 1) CS0684</span></span>
<span data-ttu-id="88db6-103">Interface 'interface' marquée avec 'CoClassAttribute' et non avec 'ComImportAttribute'</span><span class="sxs-lookup"><span data-stu-id="88db6-103">'interface' interface marked with 'CoClassAttribute' not marked with 'ComImportAttribute'</span></span>  
  
 <span data-ttu-id="88db6-104">Si vous spécifiez **CoClassAttribute** sur une interface, vous devez également spécifier **ComImportAttribute**.</span><span class="sxs-lookup"><span data-stu-id="88db6-104">If you specify **CoClassAttribute** on an interface, you must also specify **ComImportAttribute**.</span></span>  
  
 <span data-ttu-id="88db6-105">L’exemple suivant génère l’avertissement CS0684 :</span><span class="sxs-lookup"><span data-stu-id="88db6-105">The following sample generates CS0684:</span></span>  
  
```  
// CS0684.cs  
// compile with: /W:1  
using System;  
using System.Runtime.InteropServices;  
  
[CoClass(typeof(C))] // CS0684  
// try the following line instead  
// [CoClass(typeof(C)), ComImport]  
interface I  
{  
}  
  
class C  
{  
   static void Main() {}  
}  
```
