 <span data-ttu-id="a38d0-104">Une méthode avec un type de retour [void](../../csharp/language-reference/keywords/void.md) ne peut pas retourner de valeur.</span><span class="sxs-lookup"><span data-stu-id="a38d0-104">A method with a [void](../../csharp/language-reference/keywords/void.md) return type cannot return a value.</span></span> <span data-ttu-id="a38d0-105">Pour plus d’informations, consultez [Méthodes](../../csharp/programming-guide/classes-and-structs/methods.md).</span><span class="sxs-lookup"><span data-stu-id="a38d0-105">For more information, see [Methods](../../csharp/programming-guide/classes-and-structs/methods.md).</span></span>  
  
 <span data-ttu-id="a38d0-106">L’exemple suivant génère l’erreur CS0127 :</span><span class="sxs-lookup"><span data-stu-id="a38d0-106">The following sample generates CS0127:</span></span>  
  
```  
// CS0127.cs

   {  
      public int hiddenMember2  
      {

            return 0;  
         }

            return 0;   // remove return statement to resolve this CS0127  
         }  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
