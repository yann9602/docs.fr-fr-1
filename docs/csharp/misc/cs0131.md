 <span data-ttu-id="ca17e-104">Dans une instruction d’assignation, la valeur de la partie droite est assignée à la partie gauche.</span><span class="sxs-lookup"><span data-stu-id="ca17e-104">In an assignment statement, the value of the right-hand side is assigned to the left-hand side.</span></span> <span data-ttu-id="ca17e-105">La partie gauche doit être une variable, une propriété ou un indexeur.</span><span class="sxs-lookup"><span data-stu-id="ca17e-105">The left-hand side must be a variable, property, or indexer.</span></span>

## <a name="example"></a><span data-ttu-id="ca17e-108">Exemple</span><span class="sxs-lookup"><span data-stu-id="ca17e-108">Example</span></span>  
 <span data-ttu-id="ca17e-109">L’exemple suivant génère l’erreur CS0131.</span><span class="sxs-lookup"><span data-stu-id="ca17e-109">The following sample generates CS0131.</span></span>  
  
```  
// CS0131.cs  
public class MyClass  
{  
    public int i = 0;  
    public void MyMethod()  
    {  
        i++ = 1;   // CS0131  
        // try the following line instead  
        // i = 1;  
    }  
    public static void Main() { }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="ca17e-110">Exemple</span><span class="sxs-lookup"><span data-stu-id="ca17e-110">Example</span></span>  
 <span data-ttu-id="ca17e-111">Cette erreur peut également se produire si vous essayez d’exécuter des opérations arithmétiques dans la partie gauche d’un opérateur d’assignation, comme dans l’exemple suivant.</span><span class="sxs-lookup"><span data-stu-id="ca17e-111">This error can also occur if you attempt to perform arithmetic operations on the left hand side of an assignment operator, as in the following example.</span></span>

{  
    public static int Main()  
    {  
        int a = 1, b = 2, c = 3;

    }
