 <span data-ttu-id="55d2d-104">Une classe [sealed](../../csharp/language-reference/keywords/sealed.md) ne peut pas introduire un membre [protected](../../csharp/language-reference/keywords/protected.md) , car aucune autre classe ne pourra hériter de la classe `sealed` et utiliser le membre `protected` .</span><span class="sxs-lookup"><span data-stu-id="55d2d-104">A [sealed](../../csharp/language-reference/keywords/sealed.md) class cannot introduce a [protected](../../csharp/language-reference/keywords/protected.md) member because no other class will be able to inherit from the `sealed` class and use the `protected` member.</span></span>

```  
{  
   protected int i;   // CS0628

   public static void Main()  
   {  
   }  
}  
```
