## <a name="example"></a><span data-ttu-id="4f85d-110">Exemple</span><span class="sxs-lookup"><span data-stu-id="4f85d-110">Example</span></span>  
 <span data-ttu-id="4f85d-111">L’exemple suivant génère l’erreur CS0546, car la classe de base ne déclare pas un accesseur set pour la propriété.</span><span class="sxs-lookup"><span data-stu-id="4f85d-111">The following sample generates CS0546 because the base class does not declare a set accessor for the property.</span></span>  
  
```  
// CS0546.cs  
// compile with: /target:library  
public class a  
{  
   public virtual int i  
   {  
      get  
      {  
         return 0;  
      }  
   }  
  
   public virtual int i2  
   {  
      get  
      {  
         return 0;  
      }  
  
      set {}  
   }  
}  
  
public class b : a  
{  
   public override int i  
   {  
      set {}   // CS0546 error no set  
   }  
  
   public override int i2  
   {  
      set {}   // OK  
   }  
}  
```
