<span data-ttu-id="d8e52-103">Le membre conditionnel 'member' ne peut pas implémenter le membre d’interface 'base class member' dans le type 'Type Name'</span><span class="sxs-lookup"><span data-stu-id="d8e52-103">Conditional member 'member' cannot implement interface member 'base class member' in type 'Type Name'</span></span>  
  
 <span data-ttu-id="d8e52-104">Le [conditionnelle](http://msdn.microsoft.com/library/e1c4913b-74d0-421a-8a6d-c14b3f0e68fb) attribut ne peut pas être utilisé sur l’implémentation d’une interface.</span><span class="sxs-lookup"><span data-stu-id="d8e52-104">The [Conditional](http://msdn.microsoft.com/library/e1c4913b-74d0-421a-8a6d-c14b3f0e68fb) attribute cannot be used on the implementation of an interface.</span></span>  
  
 <span data-ttu-id="d8e52-105">L’exemple suivant génère l’erreur CS0629 :</span><span class="sxs-lookup"><span data-stu-id="d8e52-105">The following sample generates CS0629:</span></span>  
  
```  
// CS0629.cs  
interface MyInterface  
{  
   void MyMethod();  
}  
  
public class MyClass : MyInterface  
{  
   [System.Diagnostics.Conditional("debug")]  
   public void MyMethod()    // CS0629, remove the Conditional attribute  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
