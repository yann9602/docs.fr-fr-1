 <span data-ttu-id="6d9e5-104">Cet avertissement peut se produire dans plusieurs scénarios dans lesquels le compilateur peut vérifier qu’une variable n’est jamais référencée :</span><span class="sxs-lookup"><span data-stu-id="6d9e5-104">This warning can occur in several scenarios in which the compiler can verify that a variable is never referenced:</span></span>

-   <span data-ttu-id="6d9e5-112">Des valeurs de constante sont assignées à un champ privé ou interne dans plusieurs emplacements, mais ce champ ne fait l’objet d’aucun autre accès.</span><span class="sxs-lookup"><span data-stu-id="6d9e5-112">A private or internal field is assigned constant values in multiple locations but not otherwise accessed.</span></span> <span data-ttu-id="6d9e5-113">Si vous n’avez pas besoin de ce champ, envisagez de le supprimer.</span><span class="sxs-lookup"><span data-stu-id="6d9e5-113">If you do not need the field, consider removing it.</span></span> <span data-ttu-id="6d9e5-114">Sinon, utilisez-le de manière appropriée.</span><span class="sxs-lookup"><span data-stu-id="6d9e5-114">Otherwise, use it in some appropriate way.</span></span>

   private int i = 1;  // CS0414
