 <span data-ttu-id="bac20-104">Cette erreur se produit quand un type dans un fichier source est en conflit avec un espace de noms dans un autre fichier source.</span><span class="sxs-lookup"><span data-stu-id="bac20-104">This error occurs when a type in a source file conflicts with a namespace in another source file.</span></span> <span data-ttu-id="bac20-105">Cela se produit généralement quand le type et/ou l’espace de noms proviennent d’un module ajouté.</span><span class="sxs-lookup"><span data-stu-id="bac20-105">This typically happens when one or both come from an added module.</span></span> <span data-ttu-id="bac20-106">Pour résoudre cette erreur, renommez le type ou l’espace de noms à l’origine du conflit.</span><span class="sxs-lookup"><span data-stu-id="bac20-106">To resolve, rename the type or the namespace that caused the conflict.</span></span>  
  
 <span data-ttu-id="bac20-107">L’exemple suivant génère l’erreur CS0438 :</span><span class="sxs-lookup"><span data-stu-id="bac20-107">The following example generates CS0438:</span></span>  
  
 <span data-ttu-id="bac20-108">Compilez d’abord ce fichier :</span><span class="sxs-lookup"><span data-stu-id="bac20-108">Compile this file first:</span></span>  
  
```  
// CS0438_1.cs  
// compile with: /target:module  
public class Util  
{  
   public class A { }  
}  
```  
  
 <span data-ttu-id="bac20-109">Compilez ensuite ce fichier :</span><span class="sxs-lookup"><span data-stu-id="bac20-109">Then compile this file:</span></span>  
  
```

// compile with: /target:module  
namespace Util   
{  
   public class A { }  
}  
```  
  
 <span data-ttu-id="bac20-110">Enfin, compilez ce fichier :</span><span class="sxs-lookup"><span data-stu-id="bac20-110">And then compile this file:</span></span>

```  
// compile with: /addmodule:CS0438_1.netmodule /addmodule:CS0438_2.netmodule  
using System;  
public class Test  
{  
   public static void Main() {  
      Console.WriteLine(typeof(Util.A));   // CS0438  
   }  
}  
```
