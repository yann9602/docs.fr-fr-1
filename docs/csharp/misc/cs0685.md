# <a name="compiler-error-cs0685"></a><span data-ttu-id="52075-102">Erreur du compilateur CS0685</span><span class="sxs-lookup"><span data-stu-id="52075-102">Compiler Error CS0685</span></span>
<span data-ttu-id="52075-103">Le membre conditionnel 'member' ne peut pas avoir un paramètre out</span><span class="sxs-lookup"><span data-stu-id="52075-103">Conditional member 'member' cannot have an out parameter</span></span>  
  
 <span data-ttu-id="52075-104">Lors de l’utilisation de l’attribut <xref:System.Diagnostics.ConditionalAttribute> sur une méthode, cette dernière ne peut pas avoir de paramètre out (de sortie).</span><span class="sxs-lookup"><span data-stu-id="52075-104">When using the <xref:System.Diagnostics.ConditionalAttribute> attribute on a method, that method may not have an out parameter.</span></span> <span data-ttu-id="52075-105">En effet, la valeur de la variable utilisée pour le paramètre out n’est pas définie au cas où l’appel de la méthode ne serait compilé vers aucun élément.</span><span class="sxs-lookup"><span data-stu-id="52075-105">This is because the value of the variable used for the out parameter would not be defined in the case that the method call is compiled to nothing.</span></span> <span data-ttu-id="52075-106">Pour éviter cette erreur, supprimez le paramètre out de la déclaration de méthode conditionnelle, ou n’utilisez pas l’attribut Conditional.</span><span class="sxs-lookup"><span data-stu-id="52075-106">To avoid this error, remove the out parameter from the conditional method declaration, or don't use the Conditional Attribute.</span></span>  
  
## <a name="example"></a><span data-ttu-id="52075-107">Exemple</span><span class="sxs-lookup"><span data-stu-id="52075-107">Example</span></span>  
 <span data-ttu-id="52075-108">L’exemple suivant génère l’erreur CS0685 :</span><span class="sxs-lookup"><span data-stu-id="52075-108">The following sample generates CS0685:</span></span>  
  
```

{  
    [Conditional("DEBUG")]

        i = 1;  
    }

```
