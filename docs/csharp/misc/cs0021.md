# <a name="compiler-error-cs0021"></a><span data-ttu-id="19182-102">Erreur du compilateur CS0021</span><span class="sxs-lookup"><span data-stu-id="19182-102">Compiler Error CS0021</span></span>

 <span data-ttu-id="19182-104">L’utilisateur a tenté d’accéder à une valeur via un indexeur dans un type de données qui ne prend pas en charge [Indexers](../../csharp/programming-guide/indexers/index.md).</span><span class="sxs-lookup"><span data-stu-id="19182-104">An attempt was made to access a value through an indexer on a data type that does not support [Indexers](../../csharp/programming-guide/indexers/index.md).</span></span>  
  
 <span data-ttu-id="19182-105">L’erreur CS0021 peut se produire quand vous tentez d’utiliser un indexeur dans un assembly C++.</span><span class="sxs-lookup"><span data-stu-id="19182-105">You may get CS0021 when trying to use an indexer in a C++ assembly.</span></span> <span data-ttu-id="19182-106">Dans ce cas, décorez la classe C++ avec l’attribut `DefaultMember` pour que le compilateur C# puisse connaître l’indexeur par défaut.</span><span class="sxs-lookup"><span data-stu-id="19182-106">In this case, decorate the C++ class with the `DefaultMember` attribute so the C# compiler knows which indexer is the default.</span></span> <span data-ttu-id="19182-107">L’exemple suivant génère l’erreur CS0021 :</span><span class="sxs-lookup"><span data-stu-id="19182-107">The following sample generates CS0021.</span></span>  
  
## <a name="example"></a><span data-ttu-id="19182-108">Exemple</span><span class="sxs-lookup"><span data-stu-id="19182-108">Example</span></span>  
 <span data-ttu-id="19182-109">Ce fichier est compilé en un fichier .dll (avec l’attribut `DefaultMember` commenté), afin de générer l’erreur.</span><span class="sxs-lookup"><span data-stu-id="19182-109">This file compiles to a .dll file—with the `DefaultMember` attribute commented out—in order to generate the error.</span></span>  
  
```  
// CPP0021.cpp  
// compile with: /clr /LD  
using namespace System::Reflection;  
// Uncomment the following line to resolve  
//[DefaultMember("myItem")]  
public ref class MyClassMC  
{  
        public:  
        property int myItem[int]  
        {  
            int get(int i){  return 5; }  
            void set(int i, int value) {}  
        }  
};  
```
