 <span data-ttu-id="76a4e-104">Les méthodes qui diffèrent uniquement par leur utilisation de [ref](../../csharp/language-reference/keywords/ref.md) et de [out](../../csharp/language-reference/keywords/out.md) sur un paramètre ne sont pas autorisées.</span><span class="sxs-lookup"><span data-stu-id="76a4e-104">Methods that differ only on their use of [ref](../../csharp/language-reference/keywords/ref.md) and [out](../../csharp/language-reference/keywords/out.md) on a parameter are not allowed.</span></span>

```

   public static void Main()  
   {  
   }

   }
```
