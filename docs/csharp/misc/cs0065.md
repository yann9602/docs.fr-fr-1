# <a name="compiler-error-cs0065"></a><span data-ttu-id="8721c-102">Erreur du compilateur CS0065</span><span class="sxs-lookup"><span data-stu-id="8721c-102">Compiler Error CS0065</span></span>
<span data-ttu-id="8721c-103">'événement' : la propriété event doit posséder à la fois des accesseurs add et remove</span><span class="sxs-lookup"><span data-stu-id="8721c-103">'event' : event property must have both add and remove accessors</span></span>  
  
 <span data-ttu-id="8721c-104">Un événement qui n’est pas un champ doit avoir les deux méthodes d’accès.</span><span class="sxs-lookup"><span data-stu-id="8721c-104">An event that is not a field must have both access methods.</span></span>  
  
 <span data-ttu-id="8721c-105">L’exemple suivant génère l’erreur CS0065 :</span><span class="sxs-lookup"><span data-stu-id="8721c-105">The following sample generates CS0065:</span></span>  
  
```  
// CS0065.cs  
using System;  
public delegate void Eventhandler(object sender, int e);  
public class MyClass  
{  
   public event EventHandler Click   // CS0065,  
   {  
      // to fix, uncomment the add and remove definitions  
      /*  
      add  
      {  
         Click += value;  
      }  
      remove  
      {  
         Click -= value;  
      }  
      */  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
