## <a name="abstract-base-classes"></a><span data-ttu-id="59768-137">Classes de base abstraites</span><span class="sxs-lookup"><span data-stu-id="59768-137">Abstract Base Classes</span></span>

## <a name="interfaces"></a><span data-ttu-id="59768-145">Interfaces</span><span class="sxs-lookup"><span data-stu-id="59768-145">Interfaces</span></span>
