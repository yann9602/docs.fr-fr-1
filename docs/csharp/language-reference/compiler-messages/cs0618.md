# <a name="compiler-warning-level-2-cs0618"></a><span data-ttu-id="2dd53-102">Avertissement du compilateur (niveau 2) CS0618</span><span class="sxs-lookup"><span data-stu-id="2dd53-102">Compiler Warning (level 2) CS0618</span></span>
<span data-ttu-id="2dd53-103">'membre' est obsolète : 'texte'</span><span class="sxs-lookup"><span data-stu-id="2dd53-103">'member' is obsolete: 'text'</span></span>  
  
 <span data-ttu-id="2dd53-104">Un membre de classe a été marqué avec l’attribut `Obsolete` de sorte qu’un avertissement est émis quand le membre de classe est référencé.</span><span class="sxs-lookup"><span data-stu-id="2dd53-104">A class member was marked with the `Obsolete` attribute, such that a warning will be issued when the class member is referenced.</span></span> <span data-ttu-id="2dd53-105">Pour plus d’informations, consultez [Attributs courants](../../programming-guide/concepts/attributes/common-attributes.md).</span><span class="sxs-lookup"><span data-stu-id="2dd53-105">For more information, see [Common Attributes](../../programming-guide/concepts/attributes/common-attributes.md).</span></span>  
  
 <span data-ttu-id="2dd53-106">L’exemple suivant génère l’avertissement CS0618 :</span><span class="sxs-lookup"><span data-stu-id="2dd53-106">The following sample generates CS0618:</span></span>  
  
```csharp  
// CS0618.cs  
// compile with: /W:2  
using System;  
  
public class C  
{  
   [Obsolete("Use newMethod instead", false)]   // warn if referenced  
   public static void m2()  
   {  
   }  
  
   public static void newMethod()  
   {  
   }  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
      C.m2();  // CS0618  
   }  
}  
```
