# <a name="compiler-warning-level-1-cs1762"></a><span data-ttu-id="ef5cc-102">Avertissement du compilateur (niveau 1) CS1762</span><span class="sxs-lookup"><span data-stu-id="ef5cc-102">Compiler Warning (level 1) CS1762</span></span>
<span data-ttu-id="ef5cc-103">Une référence a été créée pour l’assembly d’interopérabilité incorporé '\<assembly1>' en raison d’une référence indirecte à cet assembly à partir de l’assembly '\<assembly2>'.</span><span class="sxs-lookup"><span data-stu-id="ef5cc-103">A reference was created to embedded interop assembly '\<assembly1>' because of an indirect reference to that assembly from assembly '\<assembly2>'.</span></span> <span data-ttu-id="ef5cc-104">Modifiez la propriété 'Incorporer les types interop' pour l’un des assemblys.</span><span class="sxs-lookup"><span data-stu-id="ef5cc-104">Consider changing the 'Embed Interop Types' property on either assembly.</span></span>  
  
 <span data-ttu-id="ef5cc-105">Vous avez ajouté une référence à un assembly (assembly1) avec une valeur de propriété `Embed Interop Types` égale à `True`.</span><span class="sxs-lookup"><span data-stu-id="ef5cc-105">You have added a reference to an assembly (assembly1) that has the `Embed Interop Types` property set to `True`.</span></span> <span data-ttu-id="ef5cc-106">Cela indique au compilateur d’incorporer les informations de type interop à partir de cet assembly.</span><span class="sxs-lookup"><span data-stu-id="ef5cc-106">This instructs the compiler to embed interop type information from that assembly.</span></span> <span data-ttu-id="ef5cc-107">Toutefois, le compilateur ne peut pas incorporer les informations de type interop à partir de cet assembly, car un autre assembly que vous avez référencé (assembly2) fait également référence à cet assembly (assembly1) et a la propriété `Embed Interop Types` définie sur `False`.</span><span class="sxs-lookup"><span data-stu-id="ef5cc-107">However, the compiler cannot embed interop type information from that assembly because another assembly that you have referenced (assembly2) also references that assembly (assembly1) and has the `Embed Interop Types` property set to `False`.</span></span>  
  
> [!NOTE]
>  <span data-ttu-id="ef5cc-108">L’affectation de la valeur `True` à la propriété `Embed Interop Types` d’une référence d’assembly revient à référencer l’assembly en utilisant l’option `/link` pour le compilateur de ligne de commande.</span><span class="sxs-lookup"><span data-stu-id="ef5cc-108">Setting the `Embed Interop Types` property on an assembly reference to `True` is equivalent to referencing the assembly by using the `/link` option for the command-line compiler.</span></span>
