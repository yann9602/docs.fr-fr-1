# <a name="compiler-error-cs1921"></a><span data-ttu-id="5aed7-102">Erreur du compilateur CS1921</span><span class="sxs-lookup"><span data-stu-id="5aed7-102">Compiler Error CS1921</span></span>
<span data-ttu-id="5aed7-103">La méthode surchargée correspondant le mieux à 'method' n’a pas la bonne signature pour l’élément initialiseur.</span><span class="sxs-lookup"><span data-stu-id="5aed7-103">The best overloaded method match for 'method' has wrong signature for the initializer element.</span></span> <span data-ttu-id="5aed7-104">La méthode Add initialisable doit être une méthode d’instance accessible.</span><span class="sxs-lookup"><span data-stu-id="5aed7-104">The initializable Add must be an accessible instance method.</span></span>  
  
 <span data-ttu-id="5aed7-105">Cette erreur est générée quand vous tentez d’utiliser un initialiseur de collection avec une classe ne disposant pas de méthode `Add` non statique publique.</span><span class="sxs-lookup"><span data-stu-id="5aed7-105">This error is generated when you try to use a collection initializer with a class that has no public non-static `Add` method.</span></span> <span data-ttu-id="5aed7-106">Si la méthode `Add` n’est pas accessible en raison de son niveau de protection (`private`, `protected`, `internal`), l’erreur CS0122 est générée, indiquant que la méthode est probablement définie comme `static`.</span><span class="sxs-lookup"><span data-stu-id="5aed7-106">If the `Add` method is not accessible because of its protection level (`private`, `protected`, `internal`) then you will get CS0122, so that this error probably means that the method is defined as `static`.</span></span>
