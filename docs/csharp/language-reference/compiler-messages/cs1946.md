# <a name="compiler-error-cs1946"></a><span data-ttu-id="8564e-102">Erreur du compilateur CS1946</span><span class="sxs-lookup"><span data-stu-id="8564e-102">Compiler Error CS1946</span></span>
<span data-ttu-id="8564e-103">Une expression de méthode anonyme ne peut pas être convertie en arborescence d’expressions.</span><span class="sxs-lookup"><span data-stu-id="8564e-103">An anonymous method expression cannot be converted to an expression tree.</span></span>  
  
 <span data-ttu-id="8564e-104">Une méthode anonyme représente un ensemble d’instructions, mais une arborescence d’expressions ne doit pas contenir d’instruction.</span><span class="sxs-lookup"><span data-stu-id="8564e-104">An anonymous method represents a set of statements but an expression tree must not contain a statement.</span></span> <span data-ttu-id="8564e-105">Par conséquent, une méthode anonyme ne peut pas être représentée par une arborescence d’expressions.</span><span class="sxs-lookup"><span data-stu-id="8564e-105">Therefore an anonymous method cannot be represented by an expression tree.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="8564e-106">Pour corriger cette erreur</span><span class="sxs-lookup"><span data-stu-id="8564e-106">To correct this error</span></span>  
  
1.  <span data-ttu-id="8564e-107">Remplacez la méthode anonyme par une expression lambda.</span><span class="sxs-lookup"><span data-stu-id="8564e-107">Change the anonymous method to a lambda expression.</span></span>  
  
## <a name="example"></a><span data-ttu-id="8564e-108">Exemple</span><span class="sxs-lookup"><span data-stu-id="8564e-108">Example</span></span>  
 <span data-ttu-id="8564e-109">L’exemple suivant génère l’erreur CS1946 :</span><span class="sxs-lookup"><span data-stu-id="8564e-109">The following example generates CS1946:</span></span>  
  
```csharp  
// cs1946.cs  
using System;  
    using System.Linq.Expressions;  
  
    public delegate void D();  
  
    class Test  
    {  
        static void Main()  
        {  
            Expression<D> tree = delegate() { }; //CS1946  
            // Try using a lambda expression instead.  
            // Expression<D> tree = (x) => x + 1;  
        }  
    }  
```
