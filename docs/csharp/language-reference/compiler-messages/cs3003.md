# Compiler Warning (level 1) CS3003
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

Type de « variable » non conforme CLS  
  
 Une variable [public](../../../csharp/language-reference/keywords/public.md), [protected](../../../csharp/language-reference/keywords/protected.md) ou `protected internal` doit avoir un type conforme à la spécification CLS \(Common Language Specification\).  Pour plus d'informations sur la conformité CLS, consultez [Indépendance du langage et composants indépendants du langage](../Topic/Language%20Independence%20and%20Language-Independent%20Components.md).  
  
## Exemple  
 L'exemple suivant génère l'erreur CS3003 :  
  
```  
// CS3003.cs  
  
[assembly:System.CLSCompliant(true)]  
public class a  
{  
    public ushort a1;   // CS3003, public variable  
    public static void Main()  
    {  
    }  
}  
```
