## <a name="example"></a><span data-ttu-id="a8884-110">Exemple</span><span class="sxs-lookup"><span data-stu-id="a8884-110">Example</span></span>  
 <span data-ttu-id="a8884-111">L’exemple suivant génère l’erreur CS1919, car un type de pointeur est unsafe :</span><span class="sxs-lookup"><span data-stu-id="a8884-111">The following example generates CS1919 because a pointer type is unsafe:</span></span>

```csharp  
{  
    public static int Main()  
    {

    }

```
