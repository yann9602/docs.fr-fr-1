# #endregion (r&#233;f&#233;rence C#)
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

`#endregion` marque la fin d'un bloc [\#region](../../../csharp/language-reference/preprocessor-directives/preprocessor-region.md).  Par exemple :  
  
```  
  
      #region MyClass definition  
class MyClass   
{  
    static void Main()   
    {  
    }  
}  
#endregion  
```
