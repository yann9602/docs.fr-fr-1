# Impossible de trouver le r&#233;pertoire .NET Framework&#160;: &lt;erreur&gt;
Le répertoire où sont stockés les assemblys [!INCLUDE[dnprdnshort](../../csharp/getting-started/includes/dnprdnshort-md.md)] est introuvable.  
  
 **ID d’erreur :** BC31508  
  
### Pour corriger cette erreur  
  
1.  Recompilez le programme pour voir si l'erreur se produit à nouveau.  
  
2.  Si l'erreur se produit encore, enregistrez votre travail et redémarrez [!INCLUDE[vsprvs](../../csharp/includes/vsprvs-md.md)].  
  
3.  Si l'erreur se produit à nouveau, réinstallez [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb-md.md)].  
  
4.  Si l'erreur persiste après la réinstallation, avertissez les services de support technique Microsoft.
