# <a name="cannot-find-net-framework-directory-lterrorgt"></a>Impossible de trouver le répertoire .NET Framework : &lt;erreur&gt;
Le répertoire où sont stockés les assemblys [!INCLUDE[dnprdnshort](~/includes/dnprdnshort-md.md)] est introuvable.  
  
 **ID d’erreur :** BC31508  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Recompilez le programme pour voir si l'erreur se produit à nouveau.  
  
2.  Si l'erreur se produit encore, enregistrez votre travail et redémarrez [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)].  
  
3.  Si l'erreur se produit à nouveau, réinstallez [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)].  
  
4.  Si l'erreur persiste après la réinstallation, avertissez les services de support technique Microsoft.
