# <a name="initializers-on-structure-members-are-valid-only-for-39shared39-members-and-constants"></a>Initialiseurs sur la structure de membres sont valides uniquement pour les &#39; Partagé &#39; les membres et constantes
Une variable de membre de structure a été initialisée dans le cadre de sa déclaration.  
  
 **ID d’erreur :** BC31049  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Utilisez une constante plutôt qu’une variable.  
  
-   Ajoutez un constructeur paramétrable à la structure. Exemple :  
  
    ```  
    Structure TestStruct  
       Public t As Integer  
       Sub New(ByVal Tval As Integer)  
          t = Tval  
       End Sub  
    End Structure  
    ```
