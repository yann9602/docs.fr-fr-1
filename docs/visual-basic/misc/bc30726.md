# <a name="39with39-contexts-and-statements-are-not-valid-in-debug-windows"></a>&#39; Avec &#39; les contextes et instructions ne sont pas valides dans les fenêtres de débogage
Les instructions`With` ne peuvent être utilisées que dans le code source.  
  
 **ID d’erreur :** BC30726  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez les instructions `With` du code de débogage.
