# <a name="goto-to-label-39ltlabelnamegt39-cannot-be-used-within-lambda-expressions"></a>GoTo étiquette &#39; &lt;labelname&gt;&#39; ne peut pas être utilisée dans les expressions lambda
Le corps d’une expression lambda doit être une expression qui calcule et retourne une valeur unique. Aucune instruction n’est autorisée, y compris les instructions `GoTo` .  
  
 **ID d’erreur :** BC36597
