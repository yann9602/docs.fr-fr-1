# <a name="conversion-from-39lttype1gt39-to-39lttype2gt39-cannot-occur-in-a-constant-expression-used-as-an-argument-to-an-attribute"></a><span data-ttu-id="49335-102">Conversion de &#39; &lt;type1&gt;&#39; à &#39;&lt; type2&gt;&#39; ne peut pas se produire dans une expression constante utilisée en tant qu’argument à un attribut</span><span class="sxs-lookup"><span data-stu-id="49335-102">Conversion from &#39;&lt;type1&gt;&#39; to &#39;&lt;type2&gt;&#39; cannot occur in a constant expression used as an argument to an attribute</span></span>
<span data-ttu-id="49335-103">Une expression utilisée pour un argument d’attribut est évaluée à un type de données différent de celui du paramètre d’attribut correspondant, et [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] n’autorise pas la conversion de type requise pour les arguments d’attributs.</span><span class="sxs-lookup"><span data-stu-id="49335-103">An expression used for an attribute argument evaluates to a data type different from that of the corresponding attribute parameter, and [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] does not allow the required type conversion for attribute arguments.</span></span>  
  
 <span data-ttu-id="49335-104">Un attribut fournit des métadonnées pour l’élément auquel il est appliqué, et le compilateur doit pouvoir construire toutes les métadonnées au moment de la compilation.</span><span class="sxs-lookup"><span data-stu-id="49335-104">An attribute provides metadata for the element it is applied to, and the compiler must be able to construct all the metadata at compile time.</span></span> <span data-ttu-id="49335-105">Pour cette raison, chaque attribut doit utiliser des valeurs constantes au moment de la compilation. Ainsi, chaque argument d’attribut doit être évalué à une valeur constante au moment de la compilation.</span><span class="sxs-lookup"><span data-stu-id="49335-105">For this reason, every attribute must use values that are constant at compile time, and therefore every attribute argument must evaluate to a compile-time constant value.</span></span>  
  
 <span data-ttu-id="49335-106">Certaines conversions de type ne peuvent pas produire de valeurs constantes au moment de la compilation.</span><span class="sxs-lookup"><span data-stu-id="49335-106">Certain type conversions cannot produce values that are constant at compile time.</span></span> <span data-ttu-id="49335-107">Par exemple, la conversion de `String` en `Double` ou `Date` dépend des paramètres régionaux au moment de l’exécution.</span><span class="sxs-lookup"><span data-stu-id="49335-107">For example, converting a `String` to a `Double` or a `Date` depends on the locale setting at run time.</span></span> <span data-ttu-id="49335-108">D’autres conversions, telles que celle d’un tableau d’un type dérivé vers un tableau de `Object`, présentent de nombreux problèmes qui ne permettent pas au compilateur de les autoriser sur les arguments d’attributs.</span><span class="sxs-lookup"><span data-stu-id="49335-108">Other conversions, such as an array of a derived type to an array of `Object`, present a variety of problems that do not permit the compiler to allow them on attribute arguments.</span></span>  
  
 <span data-ttu-id="49335-109">**ID d’erreur :** BC30934</span><span class="sxs-lookup"><span data-stu-id="49335-109">**Error ID:** BC30934</span></span>
