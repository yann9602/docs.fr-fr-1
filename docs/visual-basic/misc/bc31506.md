# <a name="39lttypenamegt39-cannot-be-used-as-an-attribute-because-it-is-declared-39mustinherit39"></a>&#39; &lt;typename&gt;&#39; ne peut pas être utilisé en tant qu’attribut, car il est déclaré &#39; MustInherit &#39;
Les classes d’attributs personnalisés ne peuvent pas être déclarées en tant que `MustInherit`.  
  
 **ID d’erreur :** BC31506  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez le modificateur `MustInherit` des attributs personnalisés.
