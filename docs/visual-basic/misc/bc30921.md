# <a name="39ltderivedtypenamegt39-cannot-inherit-from-lttypegt-39ltconstructedbasetypenamegt39-because-it-expands-the-access-of-type-39ltinternaltypenamegt39-to-ltregiongt-39ltregionnamegt39"></a><span data-ttu-id="26d87-102">&#39; &lt;nom_type_dérivé&gt;&#39; ne peut pas hériter &lt;type&gt; &#39;&lt; nom_type_base_construit&gt;&#39; car il étend l’accès de type &#39;&lt; nom_type_interne&gt;&#39; à &lt;région&gt; &#39;&lt; RegionName&gt;&#39;</span><span class="sxs-lookup"><span data-stu-id="26d87-102">&#39;&lt;derivedtypename&gt;&#39; cannot inherit from &lt;type&gt; &#39;&lt;constructedbasetypename&gt;&#39; because it expands the access of type &#39;&lt;internaltypename&gt;&#39; to &lt;region&gt; &#39;&lt;regionname&gt;&#39;</span></span>
<span data-ttu-id="26d87-103">Une classe ou une interface dérivée essaie d’étendre le niveau d’accès d’un type interne en l’utilisant comme argument de type dans une classe ou une interface de base.</span><span class="sxs-lookup"><span data-stu-id="26d87-103">A derived class or interface attempts to expand the access level of an internal type by using it as a type argument to a base class or interface.</span></span>  
  
 <span data-ttu-id="26d87-104">Le code suivant peut générer cette erreur.</span><span class="sxs-lookup"><span data-stu-id="26d87-104">The following code can generate this error.</span></span>  
  
```  
Public Class containingClass  
    Public Class baseClass(Of t)  
    End Class  
    Friend Class derivedClass  
        Inherits baseClass(Of internalStructure)  
    End Class  
    Private Structure internalStructure  
        Dim firstMember As Integer  
    End Structure  
End Class  
```  
  
 <span data-ttu-id="26d87-105">Le code à l’extérieur de `containingClass` n’est pas autorisé à accéder à `internalStructure`.</span><span class="sxs-lookup"><span data-stu-id="26d87-105">Code outside `containingClass` is not allowed to access `internalStructure`.</span></span> <span data-ttu-id="26d87-106">Toutefois, `derivedClass` est accessible à partir de n’importe quel code dans le même assembly.</span><span class="sxs-lookup"><span data-stu-id="26d87-106">However, `derivedClass` can be accessed from any code in the same assembly.</span></span> <span data-ttu-id="26d87-107">Par conséquent, `derivedClass` ne peut pas hériter de `baseClass` s’il utilise `internalStructure` comme argument de type, car cela peut exposer `internalStructure` dans l’ensemble de la région de code de définition.</span><span class="sxs-lookup"><span data-stu-id="26d87-107">Therefore, `derivedClass` cannot inherit `baseClass` if it uses `internalStructure` as a type argument, because that could expose `internalStructure` throughout the defining code region.</span></span>  
  
 <span data-ttu-id="26d87-108">**ID d’erreur :** BC30921</span><span class="sxs-lookup"><span data-stu-id="26d87-108">**Error ID:** BC30921</span></span>

-   <span data-ttu-id="26d87-110">Modifiez les niveaux d’accès des classes ou des interfaces afin que le type dérivé n’étende pas le niveau d’accès du type interne.</span><span class="sxs-lookup"><span data-stu-id="26d87-110">Adjust the access levels of the classes or interfaces so that the derived type does not expand the access level of the internal type.</span></span>  
  
     <span data-ttu-id="26d87-111">ou</span><span class="sxs-lookup"><span data-stu-id="26d87-111">-or-</span></span>
