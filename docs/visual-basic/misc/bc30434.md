# Les &#233;v&#233;nements d’un module ne peuvent pas &#234;tre d&#233;clar&#233;s &#39;&lt;sp&#233;cificateur&gt;&#39;
Vous avez utilisé un spécificateur non valide pour un événement dans une instruction `Module`. Les modules ne peuvent jamais être instanciés, ne prennent pas en charge l’héritage et ne peuvent pas implémenter d’interfaces.  
  
 **ID d’erreur :** BC30434  
  
### Pour corriger cette erreur  
  
-   Supprimez le spécificateur.
