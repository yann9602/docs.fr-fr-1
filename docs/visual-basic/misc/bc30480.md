# <a name="shared-39sub-new39-cannot-be-declared-39ltspecifiergt39"></a>Partagé &#39; Sub nouveau &#39; ne peut pas être déclaré &#39; &lt;spécificateur&gt;&#39;
Vous avez essayé d’utiliser un spécificateur dans une déclaration `Shared Sub New` , qui n’est pas valide.  
  
 **ID d’erreur :** BC30480  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez le spécificateur.
