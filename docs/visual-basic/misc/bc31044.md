# <a name="events-declared-with-an-39as39-clause-must-have-a-delegate-type"></a>Les événements déclarés avec une &#39; En tant que &#39; clause doit avoir un type délégué
Un type non délégué a été spécifié.  
  
 **ID d’erreur :** BC31044  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Supprimez la clause `As` .  
  
2.  Spécifiez un délégué avec la clause `As` .
