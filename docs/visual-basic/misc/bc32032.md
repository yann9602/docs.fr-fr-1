# &#39;End Function&#39; doit &#234;tre la premi&#232;re instruction d’une ligne
Une instruction `End Function` suit un séparateur d’instruction deux\-points \(:\).`End Function` doit être la seule instruction sur sa ligne source.  
  
 **ID d’erreur :** BC32032  
  
### Pour corriger cette erreur  
  
-   Séparez les multiples instructions en les plaçant sur des lignes différentes.
