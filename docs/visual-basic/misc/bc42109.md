# <a name="variable-39ltvariablenamegt39-is-used-before-it-has-been-assigned-a-value-visual-basic-error"></a>Variable &#39; &lt;variablename&gt;&#39; est utilisée avant qu’il a reçu une valeur (erreur Visual Basic)
Une variable est utilisée avant qu’une valeur ne lui ait été assignée, ce qui provoque une <xref:System.NullReferenceException>.  
  
 **ID d’erreur :** BC42109  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Vérifiez qu’une valeur a été attribuée à la variable.
