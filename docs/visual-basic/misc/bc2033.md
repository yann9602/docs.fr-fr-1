# Erreur d’incorporation du manifeste Win32&#160;: l’option /win32manifest est en conflit avec /nowin32manifest.
Il a été passé au compilateur Visual Basic une option de compilateur `/win32manifest` et une option de compilateur `/nowin32manifest`. Une seule de ces options peut être passée au compilateur Visual Basic.  
  
 **ID d’erreur :** BC2033  
  
### Pour corriger cette erreur  
  
1.  Supprimez l’option de compilateur `/win32manifest` ou l’option de compilateur `/nowin32manifest`.
