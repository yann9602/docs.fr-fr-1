# &#39;End AddHandler&#39; doit &#234;tre la premi&#232;re instruction sur une ligne
Une instruction `End AddHandler` suit un séparateur d’instruction deux\-points \(:\).`End AddHandler` doit être la seule instruction sur sa ligne source.  
  
 **ID d’erreur :** BC31118  
  
### Pour corriger cette erreur  
  
-   Séparez les multiples instructions en les plaçant sur des lignes différentes.
