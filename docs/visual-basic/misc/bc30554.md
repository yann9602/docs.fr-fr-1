# <a name="39ltnamespacenamegt39-is-ambiguous"></a>&#39; &lt;namespacename&gt;&#39; est ambigu
Vous avez fourni un nom qui est ambigu et donc en conflit avec un autre nom. Le compilateur [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] n’a pas de règles de résolution de conflit. Vous devez lever vous-même toute ambiguïté sur les noms.  
  
 **ID d’erreur :** BC30554  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Utilisez un nom qualifié complet.
