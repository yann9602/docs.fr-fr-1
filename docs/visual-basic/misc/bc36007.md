# <a name="39end-using39-must-be-preceded-by-a-matching-39using39"></a>&#39; À l’aide de fin &#39; doit être précédé d’une mise en correspondance &#39; À l’aide de &#39;
Une instruction `End Using` n’est précédée d’aucune déclaration `Using` correspondante.  
  
 **ID d’erreur :** BC36007  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez l’instruction `End Using` si elle est redondante.  
  
-   Fournissez la [à l’aide de l’instruction](../../visual-basic/language-reference/statements/using-statement.md) si elle est manquante.  
  
-   Déplacez l’instruction `End Using` à l’emplacement approprié dans le code.
