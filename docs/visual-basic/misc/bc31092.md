# <a name="paramarray-parameters-must-have-an-array-type"></a>Les paramètres ParamArray doivent avoir un type tableau
Des paramètres `ParamArray` ont un type non valide à cause d’un problème qui s’est produit lors de l’accès à l’assembly MSCorlib.  
  
 **ID d’erreur :** BC31092  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Recompilez le programme pour voir si l'erreur se produit à nouveau.  
  
2.  Le cas échéant, enregistrez votre travail et redémarrez [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)].  
  
3.  Si l'erreur se produit à nouveau, réinstallez [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)].  
  
4.  Si l'erreur persiste après la réinstallation, avertissez les services de support technique Microsoft.
