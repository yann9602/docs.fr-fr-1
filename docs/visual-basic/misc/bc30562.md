# <a name="39ltnamegt39-is-ambiguous-between-declarations-in-modules-39ltmodulename1gt39-and-39ltmodulename2gt39"></a>&#39; &lt;nom&gt;&#39; est ambigu entre les déclarations dans les Modules &#39;&lt; nom_module1&gt;&#39; et &#39;&lt; nom_module2&gt;&#39;
Vous avez fourni un nom qui est ambigu et donc en conflit avec un autre nom. Le compilateur [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] ne possède aucune règle de résolution de conflit ; vous devez lever vous-même toute ambiguïté sur les noms.  
  
 **ID d’erreur :** BC30562  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Utilisez un nom qualifié complet.
