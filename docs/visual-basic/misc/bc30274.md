# <a name="parameter-39ltparameternamegt39-of-39ltmethodnamegt39-already-has-a-matching-argument"></a>Paramètre &#39; &lt;nom_paramètre&gt;&#39; de &#39;&lt; MethodName&gt;&#39; a déjà un argument correspondant
Un argument est passé par nom plusieurs fois dans le même appel de procédure.  
  
 **ID d’erreur :** BC30274  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Vérifiez si l’un des noms d’arguments est mal orthographié.  
  
-   Supprimez l’argument dupliqué de la liste d’arguments.
