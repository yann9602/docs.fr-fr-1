<span data-ttu-id="b50a2-103">Un argument de type a été spécifié dans un appel à une méthode d’extension qui n’a aucun paramètre générique ou n’a aucun paramètre générique pour lequel un type n’est pas encore spécifié.</span><span class="sxs-lookup"><span data-stu-id="b50a2-103">A type argument has been specified in a call to an extension method that either has no generic parameters or has no generic parameters for which a type is not already specified.</span></span> <span data-ttu-id="b50a2-104">Par exemple, le code suivant génère cette erreur.</span><span class="sxs-lookup"><span data-stu-id="b50a2-104">For example, the following code causes this error.</span></span>  
  
```vb  
' The extension method is not generic.

    ' Body of the Sub.

```

```vb  
Dim str = "hi"  
'' The call to Example specifies a type argument.  
'' Not valid.  
'str.Example(Of String)()  
```
