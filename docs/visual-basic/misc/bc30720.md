# <a name="39on-error39-statements-are-not-valid-in-the-immediate-window"></a>&#39; sur l’erreur &#39; les instructions ne sont pas valides dans la fenêtre exécution
Vous ne pouvez pas utiliser d’instructions`On Error` dans le code source.  
  
 **ID d’erreur :** BC30720  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez les instructions `On Error` du code de débogage.
