# <a name="argument-39access39-is-not-valid-valid-values-for-output-mode-are-39openaccesswrite39-and-39openaccessdefault39"></a>Argument &#39; Accéder au &#39; n’est pas valide. Les valeurs valides pour le mode Output sont &#39;OpenAccess.Write &#39; et &#39;OpenAccess.Default &#39;
Une valeur d’accès n’est pas valide en mode Output.  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Remplacez la valeur par `OpenAccess.Write` ou `OpenAccess.Default`.
