# <a name="project-level-conditional-compilation-constant-39ltconstantnamegt39-is-not-valid-lterrorgt"></a>Constante de compilation conditionnelle au niveau du projet &#39; &lt;nom_constante&gt;&#39; n’est pas valide : &lt;erreur&gt;
Une option de compilation conditionnelle non valide est spécifiée pour le projet.  
  
 **ID d’erreur :** BC31030  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Vérifiez les constantes de compilation conditionnelle utilisées avec ce projet.
