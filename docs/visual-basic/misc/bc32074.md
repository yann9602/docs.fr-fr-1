# <a name="classes-that-are-generic-or-contained-in-a-generic-type-cannot-inherit-from-an-attribute-class"></a><span data-ttu-id="53a27-102">Les classes qui sont génériques ou contenues dans un type générique ne peuvent pas hériter d’une classe d’attributs</span><span class="sxs-lookup"><span data-stu-id="53a27-102">Classes that are generic or contained in a generic type cannot inherit from an attribute class</span></span>
<span data-ttu-id="53a27-103">Une classe générique ou imbriquée dans un type générique spécifie qu’elle hérite d’une classe d’attributs.</span><span class="sxs-lookup"><span data-stu-id="53a27-103">A class that is either generic or nested within a generic type specifies that it inherits from an attribute class.</span></span>  
  
 <span data-ttu-id="53a27-104">Visual Basic et .NET Framework ne prennent actuellement pas en charge toute combinaison d’attributs et de types génériques.</span><span class="sxs-lookup"><span data-stu-id="53a27-104">Visual Basic and the .NET Framework do not currently support any combination of attributes and generic types.</span></span> <span data-ttu-id="53a27-105">Cela signifie que les limitations suivantes s’appliquent :</span><span class="sxs-lookup"><span data-stu-id="53a27-105">This means the following limitations apply:</span></span>  
  
-   <span data-ttu-id="53a27-106">Un attribut ne peut ni être un type générique, ni être déclaré dans un type générique.</span><span class="sxs-lookup"><span data-stu-id="53a27-106">An attribute cannot be a generic type or be declared within a generic type.</span></span>  
  
-   <span data-ttu-id="53a27-107">Un attribut ne peut pas hériter d’une classe générique, et une classe générique ne peut pas non plus hériter d’un attribut.</span><span class="sxs-lookup"><span data-stu-id="53a27-107">An attribute cannot inherit from a generic class, nor can a generic class inherit from an attribute.</span></span>  
  
-   <span data-ttu-id="53a27-108">Quand vous appliquez un attribut, vous ne pouvez pas fournir un argument qui correspond à l’un des éléments suivants :</span><span class="sxs-lookup"><span data-stu-id="53a27-108">When you apply an attribute, you cannot supply an argument that is any of the following:</span></span>  
  
    -   <span data-ttu-id="53a27-109">Type générique</span><span class="sxs-lookup"><span data-stu-id="53a27-109">A generic type,</span></span>  
  
    -   <span data-ttu-id="53a27-110">Type construit à partir d’un type générique</span><span class="sxs-lookup"><span data-stu-id="53a27-110">A type constructed from a generic type,</span></span>  
  
    -   <span data-ttu-id="53a27-111">Paramètre de type d’un type conteneur</span><span class="sxs-lookup"><span data-stu-id="53a27-111">A type parameter of a containing type, or</span></span>  
  
    -   <span data-ttu-id="53a27-112">Un type construit à partir d’un paramètre de type d’un type conteneur.</span><span class="sxs-lookup"><span data-stu-id="53a27-112">A type constructed from a type parameter of a containing type.</span></span>  
  
 <span data-ttu-id="53a27-113">**ID d’erreur :** BC32074</span><span class="sxs-lookup"><span data-stu-id="53a27-113">**Error ID:** BC32074</span></span>
