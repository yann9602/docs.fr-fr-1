# Le commentaire XML ne peut pas appara&#238;tre dans une m&#233;thode ou une propri&#233;t&#233;
Un commentaire XML ne peut pas apparaître dans une méthode ni une propriété. Le commentaire XML sera ignoré. Les blocs de commentaires XML ne peuvent pas apparaître dans des méthodes ni des propriétés.  
  
 **ID d’erreur :** BC42303  
  
### Pour corriger cette erreur  
  
-   Supprimez le bloc de commentaires ou déplacez\-le vers un emplacement plus approprié.
