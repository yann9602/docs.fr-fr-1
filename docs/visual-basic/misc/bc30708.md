# <a name="loop-statements-are-not-valid-in-the-immediate-window"></a>Les instructions de boucle ne sont pas valides dans la fenêtre Exécution
Vous ne pouvez pas utiliser des instructions`Loop` dans la fenêtre **Exécution** .  
  
 **ID d’erreur :** BC30708  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez les instructions `Loop` du code de débogage.
