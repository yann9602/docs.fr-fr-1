# L’objet ne prend pas en charge cette action (Visual Basic)
Vous avez référencé une méthode ou une propriété qui n’est pas prise en charge par cet objet.  
  
### Pour corriger cette erreur  
  
1.  Pour plus d’informations, consultez la documentation de l’objet.  
  
2.  Vérifiez que les noms des propriétés et méthodes sont correctement orthographiés.
