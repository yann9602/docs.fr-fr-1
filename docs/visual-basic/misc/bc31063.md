# <a name="39set39-method-cannot-have-more-than-one-parameter"></a>&#39; Définir le &#39; méthode ne peut pas avoir plus d’un paramètre
Plusieurs paramètres sont définis pour des procédures `Set` de propriété.  
  
 **ID d’erreur :** BC31063  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez les paramètres en trop des procédures `Set` de propriété.
