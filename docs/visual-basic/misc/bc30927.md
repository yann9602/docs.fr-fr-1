# <a name="39mustoverride39-cannot-be-specified-on-39ltprocedurenamegt39-because-it-is-in-a-partial-type-that-is-declared-39notinheritable39-in-another-partial-definition"></a><span data-ttu-id="a6be8-102">&#39; MustOverride &#39; ne peut pas être spécifié sur &#39; &lt;nom_procédure&gt;&#39; car il est dans un type partiel déclaré &#39; NotInheritable &#39; dans une autre définition partielle</span><span class="sxs-lookup"><span data-stu-id="a6be8-102">&#39;MustOverride&#39; cannot be specified on &#39;&lt;procedurename&gt;&#39; because it is in a partial type that is declared &#39;NotInheritable&#39; in another partial definition</span></span>
<span data-ttu-id="a6be8-103">Une procédure ou une propriété est déclarée comme `MustOverride` dans une classe définie dans plusieurs déclarations partielles, mais l’une des définitions partielles spécifie `NotInheritable` pour la classe.</span><span class="sxs-lookup"><span data-stu-id="a6be8-103">A procedure or property is declared as `MustOverride` within a class that is defined in multiple partial declarations, but one of the partial definitions specifies `NotInheritable` for the class.</span></span>  
  
 <span data-ttu-id="a6be8-104">Quand vous divisez la définition d’une classe en plusieurs déclarations partielles, le compilateur traite la classe comme l’union de toutes ses déclarations partielles.</span><span class="sxs-lookup"><span data-stu-id="a6be8-104">When you divide the definition of a class among several partial declarations, the compiler treats the class as the union of all its partial declarations.</span></span> <span data-ttu-id="a6be8-105">Cela s’applique non seulement aux membres, mais aussi à l’implémentation, à l’héritage et au niveau d’accès.</span><span class="sxs-lookup"><span data-stu-id="a6be8-105">This applies not only to the members but also to the implementation, inheritance, and access level.</span></span>  
  
 <span data-ttu-id="a6be8-106">Pour substituer une procédure ou une propriété, une classe doit en hériter d’une classe de base.</span><span class="sxs-lookup"><span data-stu-id="a6be8-106">To override a procedure or property, a class must inherit it from a base class.</span></span> <span data-ttu-id="a6be8-107">Par conséquent, vous devez spécifier `MustOverride` pour la classe afin de spécifier `MustInherit` pour une procédure ou une propriété dans une classe de base.</span><span class="sxs-lookup"><span data-stu-id="a6be8-107">Therefore, to specify `MustOverride` for a procedure or property in a base class, you must specify `MustInherit` for the class.</span></span> <span data-ttu-id="a6be8-108">Ces mots clés étant mutuellement contradictoires, vous ne pouvez pas spécifier `MustInherit` et `NotInheritable` pour la même classe.</span><span class="sxs-lookup"><span data-stu-id="a6be8-108">Because they are mutually contradictory, you cannot specify both `MustInherit` and `NotInheritable` for the same class.</span></span>  
  
 <span data-ttu-id="a6be8-109">**ID d’erreur :** BC30927</span><span class="sxs-lookup"><span data-stu-id="a6be8-109">**Error ID:** BC30927</span></span>
