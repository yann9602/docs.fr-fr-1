# <a name="39ltnamegt39-cannot-expose-type-39lttype1gt39-used-in-the-underlying-delegate-type-39ltdelegatetypegt39-of-the-event-it-is-implementing-outside-the-project-through-ltspecifiergt-39lttype2gt39"></a>&#39; &lt;nom&gt;&#39; ne peuvent pas exposer de type &#39;&lt; type1&gt;&#39; utilisé dans le type délégué sous-jacent &#39;&lt; type_délégué&gt;&#39; de l’événement qu’il implémente en dehors du projet via &lt;spécificateur&gt; &#39;&lt; type2&gt;&#39;
Le code expose un type privé en dehors de la classe publique.  
  
 **ID d’erreur :** BC30924  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Déclarez le type `Public` ou utilisez un autre type.
