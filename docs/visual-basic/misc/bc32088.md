# <a name="type-arguments-unexpected"></a><span data-ttu-id="4d27e-102">Arguments de type inattendus</span><span class="sxs-lookup"><span data-stu-id="4d27e-102">Type arguments unexpected</span></span>
<span data-ttu-id="4d27e-103">Une clause `Implements` fournit des arguments de type pour le membre d’interface qu’elle implémente.</span><span class="sxs-lookup"><span data-stu-id="4d27e-103">An `Implements` clause supplies type arguments for the interface member it is implementing.</span></span>  
  
 <span data-ttu-id="4d27e-104">La clause `Implements` doit identifier uniquement l’interface et le membre qu’elle implémente.</span><span class="sxs-lookup"><span data-stu-id="4d27e-104">The `Implements` clause should only identify the interface and member it is implementing.</span></span> <span data-ttu-id="4d27e-105">Cela signifie que si vous déclarez une procédure générique, la clause `Of` et les arguments de type doivent apparaître dans la partie principale de la déclaration, comme ce serait le cas si vous n’implémentiez pas une procédure d’interface.</span><span class="sxs-lookup"><span data-stu-id="4d27e-105">This means that if you are declaring a generic procedure, the `Of` clause and the type arguments should appear in the main part of the declaration, just as they would if you were not implementing an interface procedure.</span></span>  
  
 <span data-ttu-id="4d27e-106">Le code suivant peut générer cette erreur.</span><span class="sxs-lookup"><span data-stu-id="4d27e-106">The following code can generate this error.</span></span>  
  
```  
Public Interface testInterface  
    Sub testSub(Of t)()  
End Interface  
Public Class testClass  
    Implements testInterface  
    Public Sub testSub() Implements testInterface.testSub(Of t)()  
    End Sub  
End Class  
```  
  
 <span data-ttu-id="4d27e-107">La déclaration qui précède la clause `Implements` doit ressembler à la définition d’interface, à l’exception d’éventuels modificateurs d’accès ou de procédure.</span><span class="sxs-lookup"><span data-stu-id="4d27e-107">The declaration preceding the `Implements` clause should look like the interface definition, except for the possible addition of access or procedure modifiers.</span></span> <span data-ttu-id="4d27e-108">Le code suivant permet d’éviter cette erreur.</span><span class="sxs-lookup"><span data-stu-id="4d27e-108">The following code avoids the error.</span></span>  
  
```  
Public Sub testSub(Of t)() Implements testInterface.testSub  
```  
  
 <span data-ttu-id="4d27e-109">**ID d’erreur :** BC32088</span><span class="sxs-lookup"><span data-stu-id="4d27e-109">**Error ID:** BC32088</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="4d27e-110">Pour corriger cette erreur</span><span class="sxs-lookup"><span data-stu-id="4d27e-110">To correct this error</span></span>  
  
-   <span data-ttu-id="4d27e-111">Supprimez la liste d’arguments de type de la clause `Implements` .</span><span class="sxs-lookup"><span data-stu-id="4d27e-111">Remove the type argument list from the `Implements` clause.</span></span>  
  
-   <span data-ttu-id="4d27e-112">Si vous implémentez un membre générique de l’interface, placez la liste d’arguments de type dans la partie principale de la déclaration, avant le mot clé `Implements` .</span><span class="sxs-lookup"><span data-stu-id="4d27e-112">If you are implementing a generic member of the interface, then put the type argument list in the main part of the declaration, preceding the `Implements` keyword.</span></span>
