# <a name="39systemruntimeinteropservicesdllimportattribute39-cannot-be-applied-to-a-sub-function-or-operator-with-a-non-empty-body"></a>&#39; System.Runtime.InteropServices.DllImportAttribute &#39; ne peut pas être appliqué à un Sub, Function ou opérateur avec un corps non vide
L’attribut `DllImportAttribute` a été appliqué à un `Sub`, `Function`ou `Operator` qui n’est pas vide.  
  
 **ID d’erreur :** BC31522  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez tout le code contenu dans le `Sub`, `Function`ou `Operator` pour utiliser cet attribut.
