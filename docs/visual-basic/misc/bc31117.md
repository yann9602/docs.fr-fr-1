# <a name="39raiseevent39-declaration-must-end-with-a-matching-39end-raiseevent39"></a>&#39; RaiseEvent &#39; déclaration doit se terminer par une correspondance &#39; End RaiseEvent &#39;
Une déclaration `RaiseEvent` doit se terminer par une instruction `End RaiseEvent` .  
  
 **ID d’erreur :** BC31117  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Vérifiez que la déclaration `RaiseEvent` se termine par une instruction `End RaiseEvent` .
