# Une expression incorpor&#233;e ne peut pas &#234;tre utilis&#233;e ici
Une expression incorporée a été incluse à un emplacement non pris en charge dans un littéral XML. Notez que les expressions incorporées ne peuvent pas être utilisées pour spécifier une valeur d’espace de noms XML.  
  
 **ID d’erreur :** BC31172  
  
### Pour corriger cette erreur  
  
-   Supprimez l’expression incorporée de l’emplacement non pris en charge.
