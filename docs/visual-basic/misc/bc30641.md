# Impossible de combiner &#39;ByVal&#39; et &#39;ByRef&#39;
Les mots clés `ByVal` et `ByRef` s’excluent mutuellement.  
  
 **ID d’erreur :** BC30641  
  
### Pour corriger cette erreur  
  
-   Spécifiez `ByVal` ou `ByRef`, mais pas les deux.
