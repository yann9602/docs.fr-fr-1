# <a name="property-39ltpropertynamegt39-with-no-parameters-cannot-be-found"></a>Propriété &#39; &lt;propertyname&gt;&#39; sans paramètre ne peut pas être trouvé
L’attribut utilisé a une propriété qui nécessite un ou plusieurs paramètres.  
  
 **ID d’erreur :** BC30658  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Spécifiez des paramètres pour la propriété.
