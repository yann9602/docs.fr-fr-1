# <a name="the-default-assembly-reference-39ltreferencegt39-has-already-been-automatically-added-ignored"></a>La référence d’assembly par défaut &#39; &lt;référence&gt;&#39; a déjà été automatiquement ajoutée ; ignoré
L’utilisateur a tenté d’ajouter la référence d’assembly par défaut, mais l’assembly a déjà été ajouté. La tentative est donc ignorée.  
  
 **ID d’erreur :** BC2024  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez le code redondant.
