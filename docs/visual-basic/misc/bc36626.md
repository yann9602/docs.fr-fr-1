# Impossible de convertir l’expression lambda en &#39;&lt;nom_type&gt;&#39;, car le type &#39;&lt;nom_type&gt;&#39; est d&#233;clar&#233; &#39;MustInherit&#39; et ne peut pas &#234;tre cr&#233;&#233;
Une instruction tente de convertir une expression lambda en un type qui peut être uniquement une classe de base et ne peut pas être utilisé pour créer une instance.  
  
 **ID d’erreur :** BC36626
