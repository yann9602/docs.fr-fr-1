# <a name="partial-method-39ltmethodnamegt39-cannot-use-the-39implements39-keyword"></a>Méthode partielle &#39; &lt;methodname&gt;&#39; ne peut pas utiliser le &#39; implémente &#39; (mot clé)
Une définition de méthode partielle ne peut pas implémenter une interface ou un membre d’interface.  
  
 **ID d’erreur :** BC31439  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Supprimez l’instruction `Implements` de la définition de méthode partielle.
