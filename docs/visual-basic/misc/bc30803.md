# <a name="39currency39-is-no-longer-a-supported-type-use-the-39decimal39-type-instead"></a>&#39; Devise &#39; n’est plus un type pris en charge ; utiliser le &#39; décimale &#39; type à la place
Le type de données `Currency` a été remplacé par le type de données `Decimal` .  
  
 **ID d’erreur :** BC30803  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Utilisez le type de données `Decimal` au lieu de `Currency`.
