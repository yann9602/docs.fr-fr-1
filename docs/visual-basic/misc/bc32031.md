# &#39;End Sub&#39; doit &#234;tre la premi&#232;re instruction sur une ligne
Une instruction `End Sub` suit un séparateur d’instruction deux\-points \(:\).`End Sub` doit être la seule instruction sur sa ligne source.  
  
 **ID d’erreur :** BC32031  
  
### Pour corriger cette erreur  
  
-   Séparez les multiples instructions en les plaçant sur des lignes différentes.
