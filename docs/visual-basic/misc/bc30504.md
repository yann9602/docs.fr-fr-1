# La propri&#233;t&#233; &#39;&lt;nom_propri&#233;t&#233;&gt;&#39; doit &#234;tre d&#233;clar&#233;e &#39;Default&#39;, car elle se substitue &#224; une propri&#233;t&#233; dans la &lt;type&gt; &#39;&lt;nom_type&gt;&#39; de base qui n’est pas une valeur par d&#233;faut
Vous avez essayé de déclarer une propriété `Default`, ce qui n’est pas une déclaration valide, car elle se substitue à une propriété qui n’a pas été déclarée `Default` dans la classe de base.  
  
 **ID d’erreur :** BC30504  
  
### Pour corriger cette erreur  
  
-   Supprimez la déclaration `Default`.
