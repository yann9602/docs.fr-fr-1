# <a name="39namespace39-statement-must-end-with-a-matching-39end-namespace39"></a>&#39; Namespace &#39; instruction doit se terminer par une correspondance &#39; Namespace de fin &#39;
Les blocs`Namespace` doivent se terminer par une instruction `End Namespace` .  
  
 **ID d’erreur :** BC30626  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Ajoutez une instruction `End Namespace` à la fin du bloc `Namespace` .
