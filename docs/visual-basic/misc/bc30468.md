# Les caract&#232;res de d&#233;claration de type ne sont pas valides dans ce contexte
Vous avez tenté d’utiliser un caractère qui n’est pas valide dans ce contexte, tel que $.  
  
 **ID d’erreur :** BC30468  
  
### Pour corriger cette erreur  
  
1.  Supprimez le caractère.
