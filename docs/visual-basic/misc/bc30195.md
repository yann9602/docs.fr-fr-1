# <a name="expected-one-of-39dim39-39const39-39public39-39private39-39protected39-39friend39-39shadows39-39readonly39-or-39shared39"></a>Attendu d’un des &#39; Dim &#39; &#39; Const &#39; &#39; Public &#39; &#39; Privé &#39; &#39; Protégé &#39; &#39; Friend &#39; &#39; Shadows &#39; &#39; En lecture seule &#39; ou &#39; Partagé &#39;
Il manque un mot clé de déclaration dans une instruction de déclaration. Il se peut qu’une déclaration d’attribut appelle une méthode.  
  
 **ID d’erreur :** BC30195  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Vérifiez si une méthode est déclarée à l’intérieur d’une déclaration d’attribut.  
  
2.  Démarrez l’instruction avec le mot clé de déclaration approprié.
