 <span data-ttu-id="79c88-104">Au moment de la compilation, [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] considère qu’une conversion prédéfinie existe à partir de n’importe quel type de référence à n’importe quel type dans sa hiérarchie d’héritage, c'est-à-dire un type dont il dérive ou qui en dérive.</span><span class="sxs-lookup"><span data-stu-id="79c88-104">At compile time, [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] considers a predefined conversion to exist from any reference type to any type in its inheritance hierarchy, that is, any type from which it derives or which derives from it.</span></span> <span data-ttu-id="79c88-105">Une telle conversion risque d’échouer au moment de l’exécution, mais le compilateur ne peut pas prédire les résultats de l’exécution, donc il permet la compilation d’une telle conversion.</span><span class="sxs-lookup"><span data-stu-id="79c88-105">Such a conversion might fail at run time, but the compiler cannot predict run-time results, so it allows any such conversion to compile.</span></span>
