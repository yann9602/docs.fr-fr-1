# <a name="unable-to-write-temporary-file-because-temporary-path-is-not-available"></a>Impossible d’écrire dans un fichier temporaire, car le chemin d’accès temporaire n’est pas disponible
[!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] n'a pas pu déterminer le chemin d'accès de l'emplacement de stockage des fichiers temporaires.  
  
 **ID d’erreur :** BC30698  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Redémarrez [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)].  
  
2.  Si le problème persiste, réinstallez [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)].
