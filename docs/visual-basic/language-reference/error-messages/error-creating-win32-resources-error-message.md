# <a name="error-creating-win32-resources-lterror-messagegt"></a>Erreur de création de ressources Win32 : &lt;message d’erreur&gt;
Le compilateur [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] appelle Assembly Linker (Al.exe, également appelé Alink) pour générer un assembly avec un manifeste. L'éditeur de liens a signalé une erreur pendant la création d'une ressource en mémoire. L'environnement peut poser problème ou votre ordinateur peut manquer de mémoire.  
  
 **ID d’erreur :** BC30136  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Examinez le message d’erreur entre guillemets et consultez la rubrique [Al.exe](../../../framework/tools/al-exe-assembly-linker.md). Pour obtenir davantage d’explications et de conseils.  
  
2.  Si l'erreur persiste, rassemblez des informations sur ses circonstances et avertissez les services de support technique Microsoft.
