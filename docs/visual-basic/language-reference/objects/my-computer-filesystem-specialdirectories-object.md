# My.Computer.FileSystem.SpecialDirectories Object
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

Fournit des propriétés pour accéder aux répertoires communément référencés.  
  
## Notes  
 Pour plus d'informations sur les méthodes et les propriétés de l'objet d' `My.Computer.FileSystem.SpecialDirectories` , consultez l' <xref:Microsoft.VisualBasic.FileIO.SpecialDirectories>.  
  
 Pour plus d'informations, consultez [How to: Retrieve the Contents of the My Documents Directory](../../../visual-basic/developing-apps/programming/drives-directories-files/how-to-retrieve-the-contents-of-the-my-documents-directory.md).  
  
## Configuration requise  
 **Espace de noms :** <xref:Microsoft.VisualBasic.MyServices>  
  
 **classe :** <xref:Microsoft.VisualBasic.MyServices.SpecialDirectoriesProxy> \(permet d'accéder à <xref:Microsoft.VisualBasic.FileIO.SpecialDirectories>\)  
  
 **Assembly :** [!INCLUDE[vbprvbruntime](../../../visual-basic/language-reference/objects/includes/vbprvbruntime_md.md)]
