# <a name="myapplicationlog-object"></a>My.Application.Log, objet
Fournit une propriété et des méthodes pour écrire les informations des événements et des exceptions dans les écouteurs de journalisation de l’application.  
  
## <a name="remarks"></a>Remarques  
 Pour plus d’informations sur les méthodes et propriétés de la `My.Application.Log` , consultez <xref:Microsoft.VisualBasic.Logging.Log>.</xref:Microsoft.VisualBasic.Logging.Log>  
  
 Pour plus d’informations, consultez [informations de journalisation de l’Application](../../../visual-basic/developing-apps/programming/log-info/logging-information-from-the-application.md).  
  
> [!NOTE]
>  Vous pouvez également utiliser les classes du .NET Framework pour enregistrer des informations provenant de votre application. Pour plus d’informations, consultez [le traçage et instrumentation d’Applications](http://msdn.microsoft.com/library/773b6fc4-9013-4322-b728-5dec7a72e743).
