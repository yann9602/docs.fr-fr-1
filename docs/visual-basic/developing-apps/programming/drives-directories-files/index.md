# Processing Drives, Directories, and Files (Visual Basic)
[!INCLUDE[vs2017banner](../../../../visual-basic/includes/vs2017banner.md)]

Vous pouvez utiliser [!INCLUDE[vbprvb](../../../../csharp/programming-guide/concepts/linq/includes/vbprvb-md.md)] pour traiter des lecteurs, des dossiers et des fichiers avec l'objet `My.Computer.FileSystem` qui offre de meilleures performances et s'avère plus facile à utiliser que les méthodes traditionnelles, telles que les fonctions `FileOpen` et `Write` \(bien qu'elles restent disponibles\).  Ces méthodes sont traitées en détail dans les sections suivantes.
