## <a name="passing-by-value"></a><span data-ttu-id="8e396-106">passer par valeur</span><span class="sxs-lookup"><span data-stu-id="8e396-106">Passing by Value</span></span>

## <a name="passing-by-reference"></a><span data-ttu-id="8e396-110">Le passage par référence</span><span class="sxs-lookup"><span data-stu-id="8e396-110">Passing by Reference</span></span>  
 <span data-ttu-id="8e396-111">Vous passez un argument *par référence* en spécifiant le [ByRef](../../../../visual-basic/language-reference/modifiers/byref.md) mot clé pour le paramètre correspondant dans la définition de procédure.</span><span class="sxs-lookup"><span data-stu-id="8e396-111">You pass an argument *by reference* by specifying the [ByRef](../../../../visual-basic/language-reference/modifiers/byref.md) keyword for the corresponding parameter in the procedure definition.</span></span> <span data-ttu-id="8e396-112">Lorsque vous utilisez ce mécanisme de passage, [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] fournit à la procédure une référence directe à l’élément de programmation sous-jacent dans le code appelant.</span><span class="sxs-lookup"><span data-stu-id="8e396-112">When you use this passing mechanism, [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] gives the procedure a direct reference to the underlying programming element in the calling code.</span></span>

 <span data-ttu-id="8e396-117">Toutefois, le mécanisme de passage et le type d’élément sont reliés entre eux.</span><span class="sxs-lookup"><span data-stu-id="8e396-117">However, the passing mechanism and element type are interrelated.</span></span> <span data-ttu-id="8e396-118">La valeur d’un type référence est un pointeur vers les données ailleurs en mémoire.</span><span class="sxs-lookup"><span data-stu-id="8e396-118">The value of a reference type is a pointer to the data elsewhere in memory.</span></span> <span data-ttu-id="8e396-119">Cela signifie que, lorsque vous passez un type référence par valeur, le code de procédure a un pointeur vers les données de l’élément sous-jacent, même s’il ne peut pas accéder à l’élément sous-jacent lui-même.</span><span class="sxs-lookup"><span data-stu-id="8e396-119">This means that when you pass a reference type by value, the procedure code has a pointer to the underlying element's data, even though it cannot access the underlying element itself.</span></span> <span data-ttu-id="8e396-120">Par exemple, si l’élément est une variable tableau, le code de procédure n’a pas accès à la variable elle-même, mais il peut accéder aux membres du groupe.</span><span class="sxs-lookup"><span data-stu-id="8e396-120">For example, if the element is an array variable, the procedure code does not have access to the variable itself, but it can access the array members.</span></span>
