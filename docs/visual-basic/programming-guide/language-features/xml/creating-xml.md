# Creating XML in Visual Basic
[!INCLUDE[vs2017banner](../../../../visual-basic/includes/vs2017banner.md)]

[!INCLUDE[vbprvb](../../../../csharp/programming-guide/concepts/linq/includes/vbprvb-md.md)] vous permet d'utiliser directement les *littéraux XML* dans votre code.  La syntaxe littérale XML, similaire à la syntaxe XML 1.0, représente des objets [!INCLUDE[sqltecxlinq](../../../../csharp/programming-guide/concepts/linq/includes/sqltecxlinq-md.md)].  Cela facilite la création par programme des éléments, documents et fragments XML puisque votre code possède la même structure que le XML final.
