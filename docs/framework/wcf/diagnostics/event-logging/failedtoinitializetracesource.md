# <a name="failedtoinitializetracesource"></a>FailedToInitializeTraceSource
ID : 101  
  
 Gravité : Erreur  
  
 Catégorie : suivi  
  
## <a name="description"></a>Description  
 La source de suivi n'a pas été initialisée. Le suivi est désactivé. L'événement répertorie l'exception, le nom de processus et l'ID de processus.
