# <a name="service-call-duration"></a>Service : durée des appels
Nom du compteur : Durée des appels  
  
## <a name="description"></a>Description  
 Durée moyenne des appels à ce service. La durée moyenne est calculée selon cette équation : (N1-N0)/(D1-D0).
