 <span data-ttu-id="86d29-107">L'interface <xref:System.ServiceModel.IExtensibleObject%601> est implémentée par les types qui permettent aux objets <xref:System.ServiceModel.IExtension%601> de personnaliser leurs fonctionnalités.</span><span class="sxs-lookup"><span data-stu-id="86d29-107">The <xref:System.ServiceModel.IExtensibleObject%601> interface is implemented by types that allow <xref:System.ServiceModel.IExtension%601> objects to customize their functionality.</span></span>
