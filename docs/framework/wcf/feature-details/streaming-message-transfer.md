-   <span data-ttu-id="63e44-106">Les transferts en flux continu exposent les messages sous forme de flux.</span><span class="sxs-lookup"><span data-stu-id="63e44-106">Streamed transfers expose the message as a stream.</span></span> <span data-ttu-id="63e44-107">Le destinataire commence à traiter les messages avant que ceux-ci ne soient complètement remis.</span><span class="sxs-lookup"><span data-stu-id="63e44-107">The receiver starts processing the message before it is completely delivered.</span></span>
