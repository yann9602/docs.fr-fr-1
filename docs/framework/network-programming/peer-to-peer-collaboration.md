## <a name="collaboration-applications"></a><span data-ttu-id="cf3ef-109">Applications de collaboration</span><span class="sxs-lookup"><span data-stu-id="cf3ef-109">Collaboration Applications</span></span>

## <a name="peer-to-peer-networking-security"></a><span data-ttu-id="cf3ef-122">Sécurité des réseaux pair à pair</span><span class="sxs-lookup"><span data-stu-id="cf3ef-122">Peer-to-Peer Networking Security</span></span>
