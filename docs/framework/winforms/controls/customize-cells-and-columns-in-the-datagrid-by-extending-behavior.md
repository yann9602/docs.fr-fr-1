<span data-ttu-id="84f98-103">Le contrôle <xref:System.Windows.Forms.DataGridView> offre plusieurs manières de personnaliser son apparence et son comportement à l'aide de propriétés, d'événements et de classes auxiliaires.</span><span class="sxs-lookup"><span data-stu-id="84f98-103">The <xref:System.Windows.Forms.DataGridView> control provides a number of ways to customize its appearance and behavior using properties, events, and companion classes.</span></span> <span data-ttu-id="84f98-104">Parfois, vous pouvez avoir des exigences pour vos cellules qui vont au-delà de ce que peuvent fournir ces fonctionnalités.</span><span class="sxs-lookup"><span data-stu-id="84f98-104">Occasionally, you may have requirements for your cells that go beyond what these features can provide.</span></span> <span data-ttu-id="84f98-105">Vous pouvez créer vos propres classes <xref:System.Windows.Forms.DataGridViewCell> pour fournir des fonctionnalités étendues.</span><span class="sxs-lookup"><span data-stu-id="84f98-105">You can create your own custom <xref:System.Windows.Forms.DataGridViewCell> class to provide extended functionality.</span></span>  
  
 <span data-ttu-id="84f98-106">Vous pouvez créer une classe <xref:System.Windows.Forms.DataGridViewCell> personnalisée en dérivant de la classe de base <xref:System.Windows.Forms.DataGridViewCell> ou de l'une de ses classes dérivées.</span><span class="sxs-lookup"><span data-stu-id="84f98-106">You create a custom <xref:System.Windows.Forms.DataGridViewCell> class by deriving from the <xref:System.Windows.Forms.DataGridViewCell> base class or one of its derived classes.</span></span> <span data-ttu-id="84f98-107">Bien que vous puissiez afficher tout type de cellule dans tout type de colonne, vous créerez en général aussi une classe <xref:System.Windows.Forms.DataGridViewColumn> personnalisée spécialisée pour l'affichage de votre type de cellule.</span><span class="sxs-lookup"><span data-stu-id="84f98-107">Although you can display any type of cell in any type of column, you will typically also create a custom <xref:System.Windows.Forms.DataGridViewColumn> class specialized for displaying your cell type.</span></span> <span data-ttu-id="84f98-108">Les classes de colonne dérivent de <xref:System.Windows.Forms.DataGridViewColumn> ou de l'un de ses types dérivés.</span><span class="sxs-lookup"><span data-stu-id="84f98-108">Column classes derive from <xref:System.Windows.Forms.DataGridViewColumn> or one of its derived types.</span></span>  
  
 <span data-ttu-id="84f98-109">Dans l'exemple de code suivant, vous allez créer une classe de cellule personnalisée nommée `DataGridViewRolloverCell` qui détecte quand la souris entre et sort des limites de la cellule.</span><span class="sxs-lookup"><span data-stu-id="84f98-109">In the following code example, you will create a custom cell class called `DataGridViewRolloverCell` that detects when the mouse enters and leaves the cell boundaries.</span></span> <span data-ttu-id="84f98-110">Pendant que la souris se trouve dans les limites de la cellule, un rectangle d'incrustation est dessiné.</span><span class="sxs-lookup"><span data-stu-id="84f98-110">While the mouse is within the cell's bounds, an inset rectangle is drawn.</span></span> <span data-ttu-id="84f98-111">Ce nouveau type dérive de <xref:System.Windows.Forms.DataGridViewTextBoxCell> et se comporte à tous autres égards comme sa classe de base.</span><span class="sxs-lookup"><span data-stu-id="84f98-111">This new type derives from <xref:System.Windows.Forms.DataGridViewTextBoxCell> and behaves in all other respects as its base class.</span></span> <span data-ttu-id="84f98-112">La classe de colonne auxiliaire se nomme `DataGridViewRolloverColumn`.</span><span class="sxs-lookup"><span data-stu-id="84f98-112">The companion column class is called `DataGridViewRolloverColumn`.</span></span>

> [!NOTE]
>  <span data-ttu-id="84f98-119">Quand vous dérivez de <xref:System.Windows.Forms.DataGridViewCell> ou <xref:System.Windows.Forms.DataGridViewColumn> et que vous ajoutez de nouvelles propriétés à la classe dérivée, veillez à substituer la méthode `Clone` pour copier les nouvelles propriétés lors des opérations de clonage.</span><span class="sxs-lookup"><span data-stu-id="84f98-119">When you derive from <xref:System.Windows.Forms.DataGridViewCell> or <xref:System.Windows.Forms.DataGridViewColumn> and add new properties to the derived class, be sure to override the `Clone` method to copy the new properties during cloning operations.</span></span> <span data-ttu-id="84f98-120">Vous devez aussi appeler la méthode `Clone` de la classe de base pour que les propriétés de la classe de base soient copiées dans la nouvelle cellule ou colonne.</span><span class="sxs-lookup"><span data-stu-id="84f98-120">You should also call the base class's `Clone` method so that the properties of the base class are copied to the new cell or column.</span></span>
