-   <span data-ttu-id="9549e-111">surcharge de méthode <xref:System.AppDomain.Load%2A?displayProperty=fullName> ou <xref:System.Reflection.Assembly.Load%2A?displayProperty=fullName> dont le premier argument est un objet <xref:System.Reflection.AssemblyName> qui identifie l’assembly à charger ;</span><span class="sxs-lookup"><span data-stu-id="9549e-111">An <xref:System.AppDomain.Load%2A?displayProperty=fullName> method overload or <xref:System.Reflection.Assembly.Load%2A?displayProperty=fullName> method overload whose first argument is an <xref:System.Reflection.AssemblyName> object that identifies the assembly to load.</span></span>
