 <span data-ttu-id="161af-106">Les classes d’exceptions définies par l’utilisateur peuvent spécifier la valeur HRESULT appropriée.</span><span class="sxs-lookup"><span data-stu-id="161af-106">User-defined exception classes can specify whatever HRESULT is appropriate.</span></span> <span data-ttu-id="161af-107">Ces classes d’exceptions peuvent modifier dynamiquement la valeur HRESULT à retourner quand l’exception est générée par la définition du champ **HResult** sur l’objet exception.</span><span class="sxs-lookup"><span data-stu-id="161af-107">These exception classes can dynamically change the HRESULT to be returned when the exception is generated by setting the **HResult** field on the exception object.</span></span> <span data-ttu-id="161af-108">Des informations supplémentaires relatives à l’exception sont fournies au client par le biais de l’interface **IErrorInfo**, qui est implémentée sur l’objet .NET dans le processus non managé.</span><span class="sxs-lookup"><span data-stu-id="161af-108">Additional information about the exception is provided to the client through the **IErrorInfo** interface, which is implemented on the .NET object in the unmanaged process.</span></span>

### <a name="to-create-a-new-exception-class-and-map-it-to-an-hresult"></a><span data-ttu-id="161af-114">Pour créer une classe d’exception et la mapper à un HRESULT</span><span class="sxs-lookup"><span data-stu-id="161af-114">To create a new exception class and map it to an HRESULT</span></span>  
  
1.  <span data-ttu-id="161af-115">Utilisez le code suivant pour créer une classe d’exception appelée `NoAccessException` et mappez-la à la valeur HRESULT `E_ACCESSDENIED`.</span><span class="sxs-lookup"><span data-stu-id="161af-115">Use the following code to create a new exception class called `NoAccessException` and map it to the HRESULT `E_ACCESSDENIED`.</span></span>

    ```cpp  
    {  
        NoAccessException () {  
        HResult = E_ACCESSDENIED;   
    }  
    }

    {  
    throw new NoAccessException();  
    }  
    ```
