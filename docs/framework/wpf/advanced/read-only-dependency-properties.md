## <a name="existing-read-only-dependency-properties"></a><span data-ttu-id="65c95-107">Propriétés de dépendance en lecture seule existantes</span><span class="sxs-lookup"><span data-stu-id="65c95-107">Existing Read-Only Dependency Properties</span></span>  
 <span data-ttu-id="65c95-108">Certaines des propriétés de dépendance définies dans le framework [!INCLUDE[TLA#tla_winclient](../../../../includes/tlasharptla-winclient-md.md)] sont en lecture seule.</span><span class="sxs-lookup"><span data-stu-id="65c95-108">Some of the dependency properties defined in the [!INCLUDE[TLA#tla_winclient](../../../../includes/tlasharptla-winclient-md.md)] framework are read-only.</span></span> <span data-ttu-id="65c95-109">En règle générale, vous définissez des propriétés de dépendance en lecture seule parce qu’elles doivent être utilisées pour déterminer des états.
